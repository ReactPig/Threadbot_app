# Release v1.0.0

## Скачать для вашей платформы:

- **Mac Apple Silicon (M1/M2/M3)**: [ThreadBot-1.0.0-arm64.dmg](ThreadBot-1.0.0-arm64.dmg)
- **Windows**: [ThreadBot Setup 1.0.0.exe](ThreadBot%20Setup%201.0.0.exe)

## Что нового

- Первый релиз ThreadBot
- AI генерация комментариев
- Анализ постов
- Помощник по написанию постов

## Установка

**Mac:** Откройте скачанный `.dmg` файл и перетащите приложение в папку Applications.

**Windows:** Запустите скачанный `.exe` файл и следуйте инструкциям установщика.
